# Projeto Alura Store

Este é um projeto desenvolvido como parte do **Programa Oracle Next Education (ONE)** em parceria com a **plataforma Alura**.

---

##  Descrição

Este projeto realiza uma análise completa das quatro lojas utilizando visualizações gráficas para identificar qual unidade apresenta o pior desempenho geral que deve ser vendida.

---

## Funcionalidades

- Faturamento por loja
- Avaliações dos clientes
- Produtos mais e menos vendidos
- Frete médio
- Categorias de maior volume
  
---

## Tecnologias utilizadas

- Google Colab
- Pandas
- Matplotlib
- Numpy
